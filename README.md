## 📁 Proyectos Destacados (TripleTen Bootcamp)

### 🚀 Proyecto 1: Pruebas Integrales, Validación de APIs y Gestión de Defectos (Sprint 4)
**Contexto del proyecto:**
Ciclo completo de pruebas enfocado en la verificación técnica y funcional de una plataforma web activa. Este proyecto integró el diseño estructurado de escenarios de prueba con la validación de componentes tanto en el frontend como en el backend a través de endpoints de API.

**Mi Análisis y Enfoque:**
* **Pruebas de API (Backend):** Utilicé **Postman** para diseñar e interactuar con colecciones de peticiones (GET, POST, PUT, DELETE), validando códigos de estado HTTP, estructuras de respuesta JSON y lógica del servidor.
* **Diseño y Control de Pruebas:** Elaboré suites detalladas de Casos de Prueba y Checklists funcionales en **Excel**, aplicando técnicas de caja negra para maximizar la cobertura.
* **Gestión de Bugs (Jira):** Documenté de forma rigurosa los errores detectados en **Jira**, asegurando reportes con pasos claros de reproducción, comportamiento esperado vs. actual, severidad y prioridad.

### 🧠 Proyecto 2: Análisis Estático de Requerimientos y Diseño Conceptual (Sprint 2)
**Contexto del proyecto:**
Fase inicial del ciclo de pruebas (STLC) enfocada en la descomposición analítica de las especificaciones del sistema. El objetivo principal fue mitigar riesgos, eliminar ambigüedades en las historias de usuario y planificar la estrategia de pruebas antes de la fase de ejecución.

**Mi Análisis y Enfoque:**
* **Modelado Visual:** Diseñé **mapas mentales y conceptuales** detallados para estructurar las funcionalidades de la aplicación, facilitando la identificación de flujos principales, alternativos y de excepción.
* **Análisis de Requisitos:** Revisión estática de la documentación del software para garantizar que los criterios de aceptación fueran testeables, claros y completos.
